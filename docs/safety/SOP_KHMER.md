# SOP – ត្រួតពិនិត្យសុវត្ថិភាពប្រចាំថ្ងៃ

## សម្រាប់អ្នកបើកបរ
1. នៅលើ Home អ្នកនឹងឃើញស្ថានភាព “ការត្រួតពិនិត្យសុវត្ថិភាពប្រចាំថ្ងៃ” ជានិច្ច។
2. ចុច “ចាប់ផ្តើមការត្រួតពិនិត្យសុវត្ថិភាព” ដើម្បីបំពេញ។
3. បំពេញជំហានទាំងអស់៖
   - ពិនិត្យយានយន្ត (ខាងក្រៅ + ម៉ាស៊ីន)
   - សុខភាពអ្នកបើកបរ
   - ឧបករណ៍សុវត្ថិភាព
   - ទំនិញ (បង្ហាញបើមាន Trip)
   - បរិស្ថាន
   - ពិនិត្យឡើងវិញ
4. ត្រូវចុចប្រអប់ “ខ្ញុំបញ្ជាក់ថាព័ត៌មាននេះត្រឹមត្រូវ” មុនដាក់ស្នើសុំអនុម័ត។
5. បន្ទាប់ពីដាក់ស្នើ ស្ថានភាព = “រង់ចាំអនុម័ត” និងមិនអាចកែបានរហូតដល់មានការសម្រេច។
6. ប្រសិនបើត្រូវបានបដិសេធ សូមអានមូលហេតុ ហើយចុច “ជួសជុល & ត្រួតពិនិត្យម្ដងទៀត” ដើម្បីបន្ត។
7. ប៊ូតុង “ចាប់ផ្តើម Trip” នឹងត្រូវបានបិទ ប្រសិនបើការត្រួតពិនិត្យសុវត្ថិភាពមិនទាន់បានអនុម័ត។

## សម្រាប់ Admin / Safety Officer
1. ចូលទៅ “ត្រួតពិនិត្យសុវត្ថិភាពប្រចាំថ្ងៃ” ក្នុង Admin Portal។
2. ប្រើ Filter ដើម្បីស្វែងរក (កាលបរិច្ឆេទ, ស្ថានភាព, ហានិភ័យ)។
3. ចូលលម្អិត ដើម្បីពិនិត្យ Item និងរូបភាពភ្ជាប់។
4. អនុម័ត៖
   - អាចកំណត់ Risk Override (Optional)
   - ស្ថានភាព = APPROVED
5. បដិសេធ៖
   - បញ្ចូលមូលហេតុ (Required)
   - ស្ថានភាព = REJECTED
6. ប្រព័ន្ធត្រូវបង្កើត Audit Log សម្រាប់សកម្មភាពទាំងអស់។

## ការទប់ស្កាត់ Dispatch
- Dispatch/Trip ត្រូវបានទប់ស្កាត់ដោយប្រព័ន្ធ ប្រសិនបើការត្រួតពិនិត្យសុវត្ថិភាពមិន APPROVED។
- សារ Khmer បង្ហាញ៖
  “មិនអាចចាប់ផ្តើម Trip បានទេ ព្រោះការត្រួតពិនិត្យសុវត្ថិភាពមិនទាន់បានអនុម័ត”

## ទិន្នន័យមេ (Admin)
1. ចូលទៅ “សុវត្ថិភាព” → “ទិន្នន័យមេ”។
2. អ្នកអាចកែប្រែ/បន្ថែម **ប្រភេទសុវត្ថិភាព** និង **ធាតុត្រួតពិនិត្យ** បាន។
3. នាំចូលពី Excel៖
   - ចូលទៅ “ធាតុត្រួតពិនិត្យសុវត្ថិភាព” ហើយចុច **នាំចូល**។
   - ឯកសារ Excel ត្រូវមាន Sheet `Category` និង `របាយការណ៍ប្រចាំថ្ងៃ`។
   - ប្រព័ន្ធនឹង Merge ទិន្នន័យដោយ `item_key = item_{id}`។
