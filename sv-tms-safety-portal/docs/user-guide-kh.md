# មគ្គុទ្ទេសក៍អ្នកប្រើប្រាស់ (ខ្មែរ)

ឯកសារនេះជាខ្លឹមសារ​រួមសម្រាប់ប្រើប្រាស់​ស្ថានីយ បន្ទាប់ពីដំនើរការ។

1. ចូលប្រើ: វាកើតមានដោយប្រើគណនី admin។
2. Dashboard: សង្ខេបស្ថានភាពសុវត្ថិភាពសម្រាប់ថ្ងៃនេះ
3. Checks: មើល និង ត្រួតពិនិត្យ daily safety checks
4. Items: កំណត់ category និង item
5. Issues: តាមដាន និង ផ្ដល់ការងារ maintenance
